# 🚀 School 21 DevOps Portfolio

Коллекция инженерных проектов, выполненных в рамках образовательной программы Школы 21 (DevOps-трек). 

Портфолио демонстрирует практические навыки системного администрирования Linux, настройки сетевой инфраструктуры, написания Bash-скриптов и развертывания систем мониторинга. Все проекты выполнены на 100% и отражают реальные задачи Junior Technical Support / L1-L2 Engineer.

---

## 🗂️ Инфраструктурные проекты

### 🐧 [DO1: Linux(Ubuntu Server)](./D01_Linux.ID_356272-2)
Развертывание и комплексное администрирование Ubuntu 20.04 Server LTS. 
- **Ключевые навыки:** Управление пользователями и правами (`sudo`, `adm`), настройка сети через `netplan` (статический IP, DNS), hardening SSH (кастомный порт 2022).
- **Диагностика:** Мониторинг ресурсов (`top`, `htop`, `df`, `du`), анализ системных логов (`/var/log/auth.log`, `syslog`), автоматизация через `cron`.

### 🌐 [DO2: Linux Network](./DO2_LinuxNetwork.ID_356275-1)
Проектирование и настройка многосегментной компьютерной сети на базе 5 виртуальных машин.
- **Ключевые навыки:** Статическая маршрутизация, настройка firewall через `iptables` (стратегии default-deny / default-allow), развертывание DHCP-сервера с резервированием по MAC.
- **Сетевые сервисы:** NAT (SNAT/DNAT), SSH-туннели (Local/Remote TCP Forwarding).
- **Диагностика:** `tcpdump`, `nmap`, `iperf3`, `traceroute`.

### 📊 [DO3: Linux Monitoring & Bash Scripting](./DO3_LinuxMonitoring_v1.0.ID_356278-1)
Разработка модульных Bash-скриптов для автоматизации сбора системных метрик и аудита.
- **Ключевые навыки:** Написание структурированного кода с валидацией ввода, парсинг конфигурационных файлов, кастомизация терминального вывода (ANSI escape codes).
- **Аудит ФС:** Рекурсивный анализ директорий, расчет MD5-хешей, статистика по типам файлов, замер времени выполнения.

### 📈 [DO4: Linux Monitoring v2.0](./DO4_LinuxMonitoring_v2.0.ID_356280-1)
Комплексное решение для генерации нагрузки, парсинга логов и мониторинга в реальном времени.
- **Ключевые навыки:** Генерация синтетических логов Nginx (combined), парсинг через `awk`, визуализация через `GoAccess`.
- **Мониторинг:** Развертывание стека `Prometheus` + `Grafana`, создание кастомных дашбордов и импорт готовых (Node Exporter).
- **Нагрузочное тестирование:** `stress`, `iperf3`, разработка собственного bash-экспортера метрик для Prometheus.
